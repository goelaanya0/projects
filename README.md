# Terminal Trading Simulator 📈

A lightweight, interactive command-line simulator written in C++ that allows users to experience virtual market trading of Bitcoin and Apple stocks with dynamically fluctuating daily prices.

## 🚀 Features
- **Dynamic Market Engine**: Prices fluctuate randomly each day using mathematical probability.
- **Inventory & Portfolio Management**: Tracks your cash balance and asset holdings in real time.
- **Transaction Validation**: Secure transaction logic prevents overdrafts and unauthorized selling.
- **Modular C++ Codebase**: Clean structure utilizing functions, loops, references, and conditional state machines.

## 🛠️ How to Compile and Run
Ensure you have a C++ compiler installed (like GCC/g++).

```bash
# Clone this repository
git clone [https://github.com/goelaanya0/projects.git](https://github.com/goelaanya0/projects.git)

# Navigate to the project directory
cd projects

# Compile the source code
g++ -o TradingSim project1.cpp

# Run the game!
./TradingSim
